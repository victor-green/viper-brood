### CONTROL FLOW
```python



value = 100


# if
if value == 100:
   print "value is 100"


# if else
if value == 101
   print "value is 101"
else:
   print "value is not 101"


# if elif else
if value == 102:
   print "value is 102"
elif value == 101:
   print "value is 101"
else:
   print "value is not 102 or 101"



```
